# keyframes.aux2

> [!WARNING]
> 超・実験的。覚悟して使ってください。

AviUtl2で擬似的にキーフレーム的な動作を実現するためのプラグイン。

## インストール

[Releases](https://github.com/sevenc-nanashi/keyframes.aux2/releases/latest)から`sevenc-nanashi.keyframes-aux2-v{{version}}.au2pkg.zip`をダウンロードし、AviUtl2のプレビューにドラッグ&ドロップしてください。
